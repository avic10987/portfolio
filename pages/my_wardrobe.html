My Wardrobe

My Wardrobe is a wardrobe visualizer program built with Python and Tkinter. I was initially inspired by this video of a simple closet automation program built with python. I browsed other existing wardrobe apps for further inspiration and decided to create a program that helps me view/organize my entire wardrobe, logs my outfits, and hosts my moodboard for viewing. The program also connects to the openWeather API to display the current local weather and saves all changes to a local JSON file so the program can be closed and re-run anytime. Overall, I found this to be a great exercise in Object Oriented Programing and coding fundamentals.
